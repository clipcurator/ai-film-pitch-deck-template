# AI Film Pitch Deck Template

English | [简体中文](README.zh-CN.md)

Pitch deck structures for AI film and short drama projects: concept, audience, visuals, workflow, budget, and release plan.

This repository is a neutral, practical resource for pitch deck and project proposal structure for AI film, AI short drama, and script-to-video projects. It is designed for teams that want repeatable decisions, clearer review, and less production guesswork.

## Why This Repository Exists

AI film projects often need to explain both story value and production method. A good pitch deck should show concept, audience promise, visual style, workflow, budget logic, release plan, and risk controls.

## Who This Is For

- creators pitching AI film ideas
- short drama teams preparing project proposals
- studios explaining AI-assisted production workflows

## Core Workflow

1. Define the input and target output
2. Score or brief the key decision points
3. Use one template to make the work reviewable
4. Run a human QA pass before publishing or production
5. Record what changed so the next cycle improves
